Описание того, что сделано и как это запустить:
1) Выполнено задание CRUD (с табличкой User), хотя изначально планировалось сделать оба задания, что отразилось на скриптах для БД
2) Скрипты для БД лежат в папке db репозитория:
	а) test_db.sql     - скрипт создания БД/схемы test
	б) root_user.sql   - скрипт создания пользователя root с паролем root и со всеми правами на табличку test
	в) user_table.sql  - скрипт создания таблицы user для задания CRUD
	г) todo_table.sql  - скрипт создания таблицы todo для задания TODOlist
	д) user_insert.sql - скрипт наполнения данных для задания CRUD
	е) todo_insert.sql - скрипт наполнения данных для задания TODOlist
	
	Для упрощения прогона скриптов по БД после деплоя приложения на Tomcat 8 можно пройти по ссылке
		localhost:8080/mvc/db-installer-step1 - Шаг 1/4. Создание БД test
		localhost:8080/mvc/db-installer-step2 - Шаг 2/4. Создание пользователя root или изменение его пароля на root
		localhost:8080/mvc/db-installer-step3 - Шаг 3/4. Создание бизнес-таблиц User и Todo
		localhost:8080/mvc/db-installer-step4 - Шаг 4/4. Наполнение бизнес-таблиц User и Todo данными
	Можно начать с любого шага, нажать на единственную кнопку и, выполнив этот шаг, перейти к следующему.
	Среди данных, которые нужно вводить на форме присутствуют: URL (jdbc:mysql://localhost), порт (3306), логин (root), пароль (root).
	В скобках указаны значения, которые вбиты по умолчанию на форме.
3) После создания БД можно запускать задание/приложение CRUD, пройдя по адресу:
		localhost:8080/mvc/crud
